# Add Project README Proposal

## Summary

Add a root README that documents project startup steps, environment configuration, and repository structure.

## Motivation

The project currently has a backend README, but the full POC spans frontend, backend, skills, prompts, OpenSpec, and docs. A root README makes local setup and project orientation easier.

## Scope

In scope:

- Root-level startup instructions for backend and frontend.
- Environment variable summary without secrets.
- Project directory structure.
- Short notes for Skill, system prompt, and OpenSpec locations.

Out of scope:

- Changing runtime behavior.
- Adding deployment documentation.
- Publishing secrets or local-only API keys.
